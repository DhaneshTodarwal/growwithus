# Performance & SEO Checklist

Core Web Vitals Targets (Google)
- LCP ≤ 2.5s
- INP ≤ 200ms
- CLS ≤ 0.1

Actions
- Image optimization (next/image), responsive sizes, lazy load below-the-fold
- CSS/JS code-split; avoid heavy client JS; use React Server Components
- Preload hero assets; edge caching (Vercel)
- Schema.org markup for Services, Blog, Organization
- sitemap.xml + robots.txt; submit in Search Console
- Mobile-first responsive design; AMP optional for blog
- Alt text, meta tags, internal linking; HTTPS enforced
