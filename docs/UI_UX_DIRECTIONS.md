# UI/UX Directions (2025)

1) Immersive Minimalism
- Clean whites/grays, blue accents; parallax/micro-animations
- ARIA labels, 4.5:1 contrast, keyboard nav

2) AI-Personalized Dark Mode
- Dark navy/black with neon accents; personalization hooks
- High-contrast toggle; screen reader friendly

3) Motion-Driven Brutalism
- Bold type, asymmetry, fluid motion; reduced motion option

Each direction has example hero, microcopy, and accessibility notes in the project brief.
