# Case Study Template

Structure (800-1200 words):
- Header: Client logo, title, industry/tags
- Challenge (~200 words)
- Approach (~300 words)
- Results (~200 words) + KPIs and testimonial
- Tech Stack Callout: "Built with Next.js, Node.js, and custom GPT integrations by ForgeVia."
- CTA: "Similar Project? Contact ForgeVia."

Samples
1) Website Development — Startup E-commerce
- Challenge: Slow non-responsive site; 25% cart abandonment
- Approach: Next.js redesign; SEO + A/B testing over 8 weeks
- Results: Load time down 60% (5s -> 2s); conversions up 35% (Tailwind, Vercel)

2) Application Development — SMB Logistics
- Challenge: Manual desktop processes; need Android/Desktop sync
- Approach: Electron desktop + Kotlin Android; 10-week sprint
- Results: Efficiency +50%; errors -40% (React Native bridge, PostgreSQL)

3) Custom AI Agents — Mid-Market Retail
- Challenge: High query volume; 20% unresolved tickets
- Approach: Chatbot with LangChain/Dialogflow; trained on 10k interactions
- Results: Resolution time -70%; CSAT 4.8/5 (Python, AWS Lambda)
